# INT-000 UNIT-003: 貸出状況・履歴確認 + アラート

## Purpose（目的）

全備品の貸出状況を一覧で把握し、返却期限に基づいたアラート表示を行う。
また全備品・全期間の貸出履歴を確認できるページを提供する。

## Boundaries（境界）

### 対象（In Scope）
- 貸出状況一覧ページ（`/status`）
  - 全備品のステータス（利用可能/貸出中）表示
  - 貸出中備品の借り手名・貸出日・返却予定日を表示
  - 返却期限アラート（超過：赤、間近1日以内：黄）
  - サマリーバッジ（利用可能件数・貸出中件数・超過件数）
- 全貸出履歴ページ（`/history`）
  - 全備品の貸出記録を日付降順で表示
  - 貸出中レコードのハイライト
- 備品一覧ページ（`/equipment`）のアラートカラー追加
  - ステータスバッジに「期限超過」「返却間近」を反映

### 対象外（Out of Scope）
- プッシュ/メール通知（PoC対象外）
- フィルタ・検索機能

## Dependencies（依存関係）

### 前提Unit
- UNIT-001・UNIT-002 完了（loansテーブルに貸出・返却データが存在すること）

### 後続Unitへの提供
- なし（最終Unit）

## 担当ユーザーストーリー
- US-007: 貸出状況の一覧確認
- US-008: 貸出履歴の確認
