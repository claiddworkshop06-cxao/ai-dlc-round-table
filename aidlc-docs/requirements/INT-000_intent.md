# INT-000 Intent

## Purpose

スマートフォンのQRコード読み取りにより、備品の貸出・返却を簡単に記録・管理できるWebアプリケーションをPoCとして構築する。

管理者が備品のマスタ登録とQRコード発行を行い、利用者がQRコードをスキャンするだけで貸出・返却操作を完結できる仕組みを実現する。また、貸出状況と履歴を一覧で確認でき、返却期限が近い備品のアラート通知によって備品の取り違えや紛失リスクを低減することを目的とする。

## Scope

### 対象
- 備品マスタの登録・編集・一覧表示（管理者機能）
- 備品ごとのQRコード生成・表示（印刷用）
- QRコードスキャンによる貸出・返却の記録（利用者機能）
- 貸出状況の一覧表示（現在どの備品が誰に貸出中かの確認）
- 貸出履歴の表示（過去の貸出・返却ログ）
- 返却期限アラート（期限超過・期限間近の備品をハイライト表示）

### 対象外
- ユーザー認証・権限管理（今回はPoC、認証なし）
- メール/プッシュ通知による外部アラート配信
- 備品の画像管理・添付ファイル
- 複数拠点・在庫数管理（数量管理）
- モバイルネイティブアプリ（Webアプリのみ）
- 外部システム連携（在庫管理システム等）
