# INT-000 Requirements

## 機能要件

### FR-001: 備品マスタ管理
- 備品を新規登録できる（備品名、説明、返却期限日数のデフォルト値）
- 備品情報を編集・削除できる
- 備品一覧を表示できる（備品名、貸出状況、登録日）

### FR-002: QRコード発行
- 備品ごとにユニークなQRコードを生成できる
- QRコードには備品IDが埋め込まれており、スキャン後に備品詳細ページへ遷移する
- QRコードを印刷用にダウンロードまたは画面表示できる

### FR-003: QRコードスキャンによる貸出・返却操作
- スマートフォンのカメラを使ってQRコードをスキャンできる
- スキャン後に該当備品の詳細ページが表示される
- 備品が「利用可能」の場合は「借りる（貸出）」ボタンを表示し、借り手名と返却予定日を入力して貸出記録を作成できる
- 備品が「貸出中」の場合は「返却する」ボタンを表示し、タップで返却記録を作成できる

### FR-004: 貸出状況の一覧表示
- 全備品の現在の貸出状況（利用可能/貸出中）を一覧で確認できる
- 貸出中の備品について、借り手名・貸出開始日・返却予定日を表示する

### FR-005: 貸出履歴の表示
- 備品ごとの貸出・返却履歴を一覧で表示できる
- 全備品の履歴を日付降順で表示できる

### FR-006: 返却期限アラート
- 返却予定日を超過している備品を視覚的にハイライト（赤）で表示する
- 返却予定日まで残り1日以内の備品を視覚的にハイライト（黄）で表示する

## 非機能要件（PoC最小限）

- Next.js 16 / React 19 / TypeScript / Tailwind CSS 4 / shadcn/ui の既存スタックを利用
- DBはDrizzle ORM + PostgreSQL (Neon) を利用
- 認証なし（PoC）
- レスポンシブ対応（スマートフォンでの操作を考慮したUI）
